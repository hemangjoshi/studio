# **App Name**: CodeShare Connect

## Core Features:

- User Authentication: Secure user authentication via Google Sign-In and Email/Password using Firebase Authentication.
- Profile Management: Users can update their display name and profile picture URL in the profile editor.
- Code Snippet Creation: Users can add code snippets with a title, code content, and a privacy toggle.
- Public Snippet Feed: Display of all public code snippets from all users.
- My Snippets: Display of the logged-in user's private and public code snippets.
- Real-time Global Chat: A single, real-time public chat room using Firestore.
- Database Security Rules: Firebase security rules to ensure data privacy and access control.

## Style Guidelines:

- Primary color: Dark Indigo (#4F46E5) to establish a sophisticated, professional tone, appropriate for developers. 
- Background color: Very dark gray (#1E1B4B), a desaturated variant of the primary hue, for a dark-themed aesthetic that's easy on the eyes.
- Accent color: Purple (#A855F7), an analogous color to the primary, but more vibrant, to create emphasis on interactive elements and important information.
- Body and headline font: 'Inter', a sans-serif font known for its modern, neutral, and readable qualities, ensuring clear communication and readability throughout the application.
- Use clean, outline-style icons to maintain a minimalist design.
- Responsive design with a persistent navigation bar for switching between Snippet Feed, Chat Room, and Profile Editor.
- Subtle transitions and animations to enhance user experience without being distracting.
